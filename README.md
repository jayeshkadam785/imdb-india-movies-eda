# IMDb India Movies EDA 🎬

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the IMDb India Movies dataset using Python. It explores movie ratings, genres, and trends through data visualization and statistical analysis.

## Features
- Data Cleaning and Preprocessing
- Rating Distribution Analysis
- Genre-wise Analysis
- Data Visualization
- Insights from IMDb India Movies Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Repository Structure
- `IMDb_India_EDA.ipynb` – Main notebook containing analysis
- `imdb-india-movies-eda.csv` – Dataset
- `CodeVedX.ipynb` – Additional notebook
